# Wiring — ESP32-C3 + 2× SG90 + 2× WS2812B ring

## Компоненты

| Компонент | Кол-во | Описание |
|---|---|---|
| ESP32-C3 dev board | 1 | Любой с 4MB flash (C3 SuperMini, DevKitC-02, etc.) |
| SG90 сервопривод | 2 | 4.8-6V, 50Hz PWM, ~250mA stall |
| WS2812B ring | 2 | По 7 LED, daisy-chain |
| Mini560 DC-DC buck | 1 | 12V → 5V, 1-2A |
| Конденсатор 470μF | 1 | Электролитический, ≥10V (для серво) |
| Конденсатор 1000μF | 1 | Электролитический, ≥10V (для LED) |
| Источник 12V | 1 | БП 12V 2A+ или 3S LiPo |
| Провода | — | ~20см сигнальные, ~30см силовые |

## Назначение пинов ESP32-C3

| Pin | Назначение | Примечание |
|---|---|---|
| GPIO2 | Servo 1 PWM | LEDC channel 0, 50Hz |
| GPIO3 | Servo 2 PWM | LEDC channel 1, 50Hz |
| GPIO10 | WS2812B DIN | RMT peripheral |
| 3.3V | — | НЕ ПИТАТЬ СЕРВО/LED |
| 5V (VBUS) | — | Только если ESP на USB |
| GND | Common ground | ⚠️ Обязательно общая с Mini560 |

## Цвета проводов SG90

| Цвет | Назначение |
|---|---|
| 🔴 Red | VCC (4.8-6V) |
| 🟤 Brown | GND |
| 🟠 Orange | Signal (PWM) |

## Цвета проводов WS2812B ring

| Pad | Назначение |
|---|---|
| VCC / 5V | Питание +5V |
| GND | Земля |
| DIN | Data In (от ESP или предыдущего кольца) |
| DOUT | Data Out (к следующему кольцу, если есть) |

## Проверка перед включением

✅ Чек-лист:

- [ ] Общая GND: ESP32-C3 GND ↔ Mini560 OUT- ↔ 12V source GND
- [ ] Mini560 IN+ ↔ 12V+, IN- ↔ 12V-
- [ ] Mini560 OUT+ = 5V (проверить мультиметром ДО подключения нагрузки)
- [ ] Конденсатор 470μF на 5V (плюс к +5V, минус к GND) — близко к серво
- [ ] Конденсатор 1000μF на 5V — близко к LED
- [ ] Servo 1: красный → +5V, коричневый → GND, оранжевый → GPIO2
- [ ] Servo 2: красный → +5V, коричневый → GND, оранжевый → GPIO3
- [ ] WS2812B ring 1: VCC → +5V, GND → GND, DIN → GPIO10
- [ ] WS2812B ring 2: VCC → +5V, GND → GND, DIN → DOUT ring 1
- [ ] **Ничего не подключено к 3.3V ESP** (кроме, возможно, level shifter)

## Если не работает

### Серво не двигается
1. Проверить PWM мультиметром на GPIO2/GPIO3 — должно быть ~1.5V (среднее между 0 и 3.3V)
2. Проверить напряжение на red проводе серво — должно быть 5V ±0.2V
3. Проверить `servo_enabled` в UI (тумблер должен быть ON)
4. Если угол = 90° — серво стоит на середине (нормально)

### LED не светятся
1. Проверить daisy chain: DOUT первого кольца → DIN второго (НЕ DOUT→DOUT!)
2. Проверить полярность конденсатора (взрывается при перепутывании)
3. Если горит только первый диод — нужен level shifter 74AHCT125 (3.3V→5V signal)
4. Проверить `led_brightness > 0` в UI

### ESP32-C3 не запускается / перезагружается
1. Brownout detection — питание проседает. Добавить ещё кондёр на 5V.
2. Проверить ток Mini560 — должен быть ≥1A
3. Отключить LED и серво, проверить что ESP стартует одна
4. Serial Monitor (115200) — должен показать "ESP32 Control Hub"

### WiFi не появляется
1. Подождать 5-10 сек после включения
2. Serial Monitor показать "AP IP: 192.168.4.1"
3. Проверить SSID `ESP32-Control` в списке WiFi сетей
4. Пароль: `12345678` (мин 8 символов)

## Level shifter (опционально)

Если WS2812B глючит при питании от 5V и сигнале 3.3V от ESP:

```
ESP32-C3 GPIO10 ──── 74AHCT125 A1
                       VCC ──── 5V
                       GND ──── GND
                       Y1 ──── WS2812B DIN
```

74AHCT125 повышает 3.3V сигнал до 5V — стабильная работа на длинных проводах.
